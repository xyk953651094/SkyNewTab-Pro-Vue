## 简介
适用于Chrome、Edge、Firefox、~~Safari~~ 的新标签页。  
基于 Html + JavaScript + Less + Vue + Webpack + Arco Design + Bootstrap Icon。   
支持语言：中文

* 天气来源：https://www.jinrishici.com
